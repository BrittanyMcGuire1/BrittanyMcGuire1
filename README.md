## 🚀 Projects

### 🥗 FridgeMate — AI Recipe Generator
> Python, FastAPI, React, PostgreSQL, Google Gemini API, Vercel, Render

An AI-powered full-stack recipe generator that creates personalized recipes based on ingredients you have on hand. Supports dietary preferences including vegan, vegetarian, gluten-free, and dairy-free with allergen filtering. Features ingredient labeling showing what you have versus what you need to buy, a strict mode toggle, and every recipe saved to a cloud PostgreSQL database.
